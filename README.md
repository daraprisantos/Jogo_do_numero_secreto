# 🎮 Jogo do Número Secreto

## 💻 Sobre o Projeto
Este é um jogo interativo de adivinhação desenvolvido como parte dos meus estudos e transição de carreira para a área de Tecnologia da Informação (Engenharia de Software). O objetivo do jogador é descobrir o número secreto gerado aleatoriamente pelo sistema.

O projeto foca na aplicação prática de lógica de programação, manipulação de elementos na tela (DOM) e interatividade com o usuário.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação da página e dos elementos do jogo.
* **CSS3:** Estilização visual e responsividade.
* **JavaScript:** Lógica por trás do sorteio, validação dos palpites e interação com os botões.
* **Vercel:** Plataforma utilizada para o deploy (hospedagem) do projeto.

## 🕹️ Como Jogar
Você pode testar o jogo diretamente no seu navegador, sem precisar baixar nada!
**👉 [Clique aqui para jogar](https://jogo-da-dara.vercel.app/)**
